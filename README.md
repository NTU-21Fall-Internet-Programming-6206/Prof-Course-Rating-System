## 目录结构说明
### 1. docs目录
所有项目文档均储存在此目录.

对于一个markdown文档 `A.md`，其图片的储存目录应当放在该文档同目录下，并明命名为 `A_image`。 例如 `docs/test.md` 的图片储存目录应当是 `docs/test_image`。


### 2. codes目录
所有代码工程会保存在此目录。
